Usage Instructions

  1. Download the appropriate executable for your system
  2. Extract the ZIP file to a folder of your choice
  3. Run: kpmg-db-solver-nonadmin.exe discover
  4. Follow the interactive prompts to configure the tool
  5. View reports in the ./reports/ folder

System Requirements

  • Windows 10/11 or Windows Server 2016+
  • Read access to Canvus Server directories (no admin privileges required)
  • Network access to Canvus Server API
  • Access to Canvus Server 3.3.0

Output

The tool generates:

  • Detailed Report: Missing assets grouped by canvas with widget information and backup locations
  • CSV Export: Comprehensive list of missing assets with backup status and file locations
  • Backup Location Report: All backup file locations for assets that can be restored
Limitations: This non-admin version provides read-only access and cannot restore assets. For asset restoration, you would need administrator privileges and the full version of the tool.
